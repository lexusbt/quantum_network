# Data Directory Structure

This directory contains datasets and processed data for the quantum network routing project.

## Directory Organization
```
data/
├── raw/               # Raw SNAP datasets (NOT in git - local only)
│   ├── ca-GrQc.txt
│   ├── email-Enron.txt
│   ├── p2p-Gnutella08.txt
│   ├── ca-CondMat.txt
│   └── ca-HepPh.txt
├── processed/         # Preprocessed graphs (NOT in git - generated)
└── qubo/             # Generated QUBO matrices (NOT in git - generated)
```

## Getting the Data

### First Time Setup (New Machine)

**The raw data files are NOT included in this repository.** You need to:

1. **Download from SNAP** (recommended):
```bash
   python scripts/download_datasets.py
```

2. **Or copy from another machine**:
   - If you have the files on another machine, copy them to `data/raw/`
   - Verify checksums: `shasum -a 256 -c data/checksums.txt`

3. **Or download manually**:
   - See `data/DATASETS.md` for download links

### Syncing Between Machines

Since data files are gitignored:
- **Option 1:** Download fresh on each machine (recommended)
- **Option 2:** Use cloud storage (Dropbox, Google Drive) to sync `data/raw/`
- **Option 3:** Use rsync/scp to copy between machines

## Why Data Files Aren't in Git

✅ Keeps repository small and fast  
✅ Easier to clone and pull  
✅ Data files don't change often  
✅ Same source of truth (SNAP) for everyone  

## Processed Data

Processed data in `data/processed/` and `data/qubo/` is generated by scripts and also gitignored. 

To regenerate processed data:
```bash
python src/data_processing/preprocess_graphs.py
```
